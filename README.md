# TEI-tools

This repository distributes files and tools introduced in the TEI learning videos.

You are also welcome to use these materials freely for other purposes if they are useful.  
**License:** CC BY

The file `meros.xml` was downloaded from the example section (“Run, Melos!”) on the **TEI Viewer for Japanese Classical Texts** provided by the TEI Study Group.

- [TEI Study Group](https://tei.dhii.jp/home)
- [TEI Viewer for Japanese Classical Texts](https://tei.dhii.jp/teiviewer4eaj)
- [Original file](https://raw.githubusercontent.com/TEIC/CETEIcean/master/tutorial_ja/meros.xml)

The version included in this repository has been modified by adjusting tags and formatting.
